# WebSite
홈페이지 외주작업 - 김덕성

## Getting started
node.js가 설치되어있어야합니다. 
npm과 bower의 dependency를 설치합니다.

> npm install && bower install

페이지를 build합니다.

> gulp build

## Dependency

개발 환경

- gulp
- browser-sync

주요 모듈

- sass (node-sass)

## Demo

진행상황은 cleanbasket.github.io/WebSite/(페이지이름.html)로 확인할 수 있습니다.

- [main](http://cleanbasket.github.io/WebSite/main.html)
